Supported tags and respective `Dockerfile` links
================================================

* [`latest` *(Dockerfile)*][1]

This build is based on the official [Debian Wheezy image][2]. We simply install some useful tools to handle data:

* [less][3]
* [vim][4]
* [wget][5]

Issues
